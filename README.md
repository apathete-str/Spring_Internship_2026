# Uncertainty-Aware Disease Risk Prediction

## Project Overview
**Problem Statement:** Uncertainty-Aware Disease Risk Prediction under Noisy and Outlier-Rich Medical Data.

This project focuses on evaluating deep learning (DL) model reliability under real-world medical data conditions—specifically addressing noise and outliers. The core objective is not merely binary disease prediction, but leveraging uncertainty estimation techniques to quantify the model's confidence in its predictions.

## Proposed Datasets
We are currently exploring secondary datasets from credible sources to finalize our specific disease focus. Prospective sources include:
* [Indian Liver Patient Dataset (ILPD)](https://archive.ics.uci.edu/ml/datasets/ILPD+(Indian+Liver+Patient+Dataset))
* [MIDAS Medical Imaging Datasets](https://midas.rutgers.edu/)
* [AIKosh Healthcare Data](https://aikosh.in/)
